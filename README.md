# Dynamic CFO & Risk Intelligence Dashboard

## Live Demo
https://123ujjawl.github.io/bajaj-finance-Management-dashboard/

## Overview
This project is a live CFO and Risk Intelligence Dashboard built for Bajaj Finance analysis. It converts financial, operating, risk and peer benchmark data into an executive-level management reporting dashboard.

## Key Features
- KPI monitoring for AUM, PAT, ROA, ROE, GNPA, NNPA, Deposits and Customers
- Variance analysis between FY25 and FY26
- Executive summary and management commentary
- Scenario analysis: Base, Bull and Bear case
- Risk heat map and financial health score
- Peer benchmarking against listed NBFCs
- Strategic insights, action tracker and investment thesis
- Google Sheets to Apps Script API integration
- GitHub Pages deployment

## Technology Stack
- Google Sheets
- Google Apps Script
- HTML
- JavaScript
- Tailwind CSS
- Chart.js
- GitHub Pages

## Architecture
Google Sheets → Apps Script API → JavaScript Processing → CFO Dashboard → Management Decision Support

## Use Case
Designed for CFO review, FP&A reporting, business performance monitoring, risk review and board-level decision support.
